# Svelte Learning Journey 🚀

Ini adalah proyek pembelajaran saya untuk menguasai **Svelte** dengan TypeScript dan Vite.

## 📚 Tentang Proyek Ini

Saya sedang dalam tahap belajar Svelte dan menggunakan template Svelte + TypeScript + Vite sebagai fondasi pembelajaran saya. Proyek ini akan terus berkembang seiring dengan kemajuan saya dalam menguasai framework ini.

---

## ✅ Checkpoint 1 - Foundation & Basic Components

### Yang telah saya pelajari dan implementasikan:

1. **Persiapan Proyek**
   - ✓ Setup proyek Svelte + TypeScript + Vite
   - ✓ Konfigurasi TypeScript dan environment
   - ✓ Memahami struktur folder proyek

2. **Komponen Dasar**
   - ✓ **Counter.svelte** - Komponen untuk menambah/mengurangi angka dengan state management
   - ✓ **Gate.svelte** - Komponen yang mendemonstrasikan conditional rendering dan props
   - ✓ **App.svelte** - Komponen utama yang mengintegrasikan semua komponen

3. **Konsep Svelte yang Dipelajari**
   - ✓ Reactive variables dan data binding
   - ✓ Event handling (onClick, handlers)
   - ✓ Component composition dan props
   - ✓ Styling dalam komponen Svelte
   - ✓ Template syntax dan directives dasar

4. **Tools & Setup**
   - ✓ VS Code + Svelte extension
   - ✓ Hot Module Replacement (HMR)
   - ✓ TypeScript integration

### Struktur Proyek:
```
src/
├── App.svelte          (Komponen utama)
├── main.ts             (Entry point)
├── app.css             (Global styles)
└── lib/
    ├── Counter.svelte  (Counter component)
    └── Gate.svelte     (Gate component)
```
